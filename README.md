# 📊 COVID-19 Analysis and Forecasting

A Data Science project that performs COVID-19 data analysis, visualization, and future case forecasting using Python and Prophet.

---

## 🚀 Project Overview

This project analyzes COVID-19 global data to understand:

- Confirmed Cases
- Death Cases
- Recovered Cases
- Active Cases

It also compares the most affected countries and predicts future confirmed cases using Facebook Prophet Time Series Forecasting.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Prophet
- Jupyter Notebook
